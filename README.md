# Sculture Api
Sculture is e-book apps and quiz about culture in Indonesia
### How to install

- Install docker and docker-compose
- Clone this repository
- Then npm install
- Next run mongo container
```bash
docker-compose up -d
```
- Last 
```
nodemon or node index.js
```
- Done, api successfull running
